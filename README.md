# Sales-Data-Analysis
Sales Data Analysis Dashboard built using Power BI to analyze business performance across regions, categories, and time. Includes interactive visuals, KPIs, and actionable insights.

Project Title : Sales Data Analysis and Visualization using Power BI
. This project focuses on analyzing sales data to evaluate overall business performance using Power BI. The dashboard provides insights into key metrics such as total sales, profit, order volume, and profit margins. It helps in understanding trends across time, regions, and product categories through interactive visualizations.

Overview
.This project focuses on analyzing sales data to evaluate business performance using an interactive dashboard built in Power BI. The objective is to transform raw transactional data into meaningful insights that support data-driven decision-making.
